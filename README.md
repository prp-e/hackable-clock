# Hackable Digital Clock 

This is a hackable [arduino](http://arduino.cc) based project, I made in my free time!
You can copy `clock.ino` file and use it to make your own digital clock. 

## Hardware 
* Arduino Uno
* 2x16 LCD and Keypad Shield
